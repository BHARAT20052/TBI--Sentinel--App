# TBI Sentinel: Defense AI for Brain Injury Detection

## Overview
Real-time AI tool for Indian Army field medics:
- **MRI Analysis:** OpenCV detects anomalies in 10 seconds
- **Vitals Forecasting:** ARIMA predicts 48-hour heart rate
- **JSON Report:** Structured output for triage
